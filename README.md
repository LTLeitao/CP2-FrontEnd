# 🔥 Dark Souls Blog - CHECKPOINT

Projeto desenvolvido para a disciplina de Front-End, com o objetivo de aplicar conceitos de HTML5 e CSS3 na criação de uma página web temática.

---

## 🎮 Sobre o Projeto

O **CHECKPOINT** é um site inspirado no universo de Dark Souls, trazendo uma ambientação sombria e imersiva.
A página apresenta informações sobre classes do jogo, o mundo de Lordran e uma seção de login simulada.

---

## 🧩 Funcionalidades

* Página temática com design inspirado em Dark Souls
* Seção de apresentação (banner)
* Exibição de classes com descrição
* Seção explicativa sobre o mundo do jogo
* Formulário de login (simulado)
* Layout responsivo básico com Flexbox

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts
* Flexbox

---

## 🎨 Recursos Utilizados

* Imagens temáticas de Dark Souls
* Tipografia personalizada (Google Fonts)
* Paleta de cores escura com destaque em tons de laranja
